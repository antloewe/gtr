Detailed Description

Represents a mixture of categorical distributions over

<V>

. A mixture of distributions is a distribution which can be interpreted as being derived from an underlying set of distributions. In this case, each of these distributions is a

Categorical

. Each of the underlying distributions has a key so that conditional probabilities are easily calculator. The respective weights of each of the underlying distributions is represented as the ratio of the counts of the particular

Categorical

with respect to the sum of all the

Categorical

frequency sums.

Parameters
<K>the type of the keys to the individual categorical distributions in the mixture
<V>the type of the support of the categorical distributions in the mixture
