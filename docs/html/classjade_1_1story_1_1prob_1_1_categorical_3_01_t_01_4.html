Detailed Description

Represents a categorical distribution over

<T>

. This discrete distribution assigns a mass to each possible outcome. This class represents these probabilities a set of frequencies. The probability of any given event is equal to its count divided by the sum of all counts.

Parameters
<T>the type of the support of the distribution
